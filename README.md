# 🎨 AI Code Review Agent – Frontend (React)

This is the **React frontend** for the AI Code Review Agent.

It allows users to:
- Enter a GitHub repo + file path
- Trigger AI analysis
- View results in tabs (Overview / Charts / Suggestions)
- See interactive charts and suggestions

---

---

## ⚙️ Requirements
- Node.js (v18 recommended)
- npm

---

## 🚀 Run Frontend Locally

### 1️⃣ Clone the repository
```bash
https://github.com/harshJyoriya1009/Code_review_frontend.git
```

### 2️⃣ Go inside frontend folder
```bash
cd Code_review_frontend
```

### 3️⃣ Install dependencies
```bash
npm install
```

### 4️⃣ Start development server
```bash
npm run dev
```
Frontend will run on:
http://localhost:5173
